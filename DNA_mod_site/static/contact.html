Contributing

Please report any DNAmod specific items or issues with citations regarding mapping techniques or natural occurrence to our issue tracker.

Please report any issues with imported ChEBI annotations to the ChEBI issue tracker.

Contact

There is a moderated dnamod-announce mailing list that you can subscribe to for information on new releases of DNAmod.

There is also a dnamod-users mailing list for general discussion and questions about the use of DNAmod.

An issue tracker is available on GitHub for submitting curation requests and for reporting any other DNAmod issues.
